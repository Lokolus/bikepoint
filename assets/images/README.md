# Bilder für BikePoint Header

## Anweisungen

Bitte platziere dein **Bild "3"** in diesem Verzeichnis mit einem der folgenden Dateinamen:

- `3.jpg` (empfohlen)
- `3.jpeg`
- `3.png`
- `3.webp`

## Empfohlene Bildspezifikationen

- **Format**: Schwarz-Weiß oder Graustufen für beste Wirkung
- **Auflösung**: Mindestens 1920x1080px (Full HD)
- **Seitenverhältnis**: 16:9 oder 16:10
- **Dateigröße**: Optimiert für Web (< 500KB empfohlen)

## Design-Konzept

Das Header-Image wird mit einem eleganten Overlay versehen:
- Dunkles Gradient-Overlay für bessere Textlesbarkeit
- Subtile Teal-Akzentfarbe (#0D9488) im Overlay
- Smooth Parallax-Effekt beim Scrollen
- Sanfte Zoom-Animation (20s Loop)

## Farbschema

Die Seite verwendet jetzt:
- **Akzentfarbe**: Teal (#0D9488) - passt perfekt zu Schwarz-Weiß
- **Hover-Akzent**: Helles Teal (#14B8A6)
- **Minimalistischer Stil**: Aufgeräumt mit feinen, smooth Effekten
